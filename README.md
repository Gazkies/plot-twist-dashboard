# 🎬 Plot Twist Analytics Dashboard

[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://plot-twist-analytics.streamlit.app/)

An interactive Streamlit dashboard analyzing movie dataset trends, profitability, and box office performances for our Information Systems and Data Analytics project. Built to turn raw cinematic data into dynamic, human-readable insights.

### 🚀 [Click Here to View the Live Dashboard](https://plot-twist-analytics.streamlit.app/)

---

## 💡 Key Features

*   **Dynamic KPIs & ROI Tracking:** Real-time metrics calculating average budget, gross revenue, and Return on Investment (ROI) based on user-selected filters.
*   **Box Office Champion Spotlight:** A dynamic "hero card" that automatically identifies and highlights the top-performing movie for any selected genre and year.
*   **Automated Data Storytelling:** Context-aware insight blocks that adapt to the data, warning users of financial losses or highlighting massive returns depending on the selected criteria.
*   **Deep-Dive Visualizations:**
    *   Interactive Correlation Heatmap (exploring relationships between Budget, Gross, Score, and Runtime).
    *   Historical Profitability & Runtime trends over the decades.
    *   Interactive Budget vs. Gross Revenue scatter plots.
